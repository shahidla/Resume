SAP Product and Custom Solution: SAP UI Logging

Originally published on SAP Blogs, Oct 2014

This article described a custom solution built around SAP UI Logging. The project involved capturing user actions and selection-screen values from SAP transaction codes and BI reports, then transforming logged data into meaningful information for external evaluation and analysis.

Project context

The solution sat on top of the SAP RCS product LOGWIN. The custom component transferred generated logs to an external log structure and mapped raw technical information back to user actions for configured transaction codes.

Engineering responsibilities

The work included custom ABAP solution design, log transformation, mapping logic, transaction-level interpretation, and evaluation support for business and audit use cases.
