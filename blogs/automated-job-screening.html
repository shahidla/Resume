Automated Job Screening Using SAP Integration Suite, Adzuna, and ChatGPT

Originally published on SAP Community, Nov 2025

This exercise explores SAP Integration Suite as an orchestration layer for profile-based job screening. Instead of relying on simple keyword matching, the prototype retrieves new SAP-related roles from Adzuna and classifies them against a defined technical profile using a ChatGPT-based classifier.

Flow overview

A scheduled integration flow calls the Adzuna API, parses the response, filters jobs already seen in previous runs, and sends each new role to the model for classification. The resulting categories are aggregated into a single HTML email with colour-coded outcomes.

Implementation pattern

  • Timer trigger starts the recurring flow.
  • Request parameters are prepared for role keywords, location, and pagination.
  • A data store tracks previously processed job ids.
  • Only unseen roles are sent to the classification model.
  • Results are formatted and sent as a consolidated summary email.

The design combines API connectivity, state management, model-based evaluation, and notification into a repeatable fetch-filter-classify-notify cycle.
