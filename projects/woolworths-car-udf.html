Retail / SAP CAR UDF / HANA

Woolworths high-volume SAP CAR UDF developments.

SAP Australia contractor, July 2022 to September 2024.

Context

Led SAP CAR UDF custom developments supporting forecast data calculations and machine-learning reporting in a high-volume retail environment.

Architecture Contribution

  • Designed scalable AMDP, CDS, and ABAP objects for partitioned data processing.
  • Processed more than 40 billion records across several partitioned tables within the same program.
  • Used Code Inspector, SQL Trace, AMDP Trace, and PlanViz to identify bottlenecks.
  • Refactored existing logic using CDS and AMDP to align with S/4HANA and cloud coding standards.

Impact

Delivered performant custom processing for very large data volumes while supporting forecast and machine-learning reporting requirements.
