S/4HANA / Planning / Fiori

Stockland S/4HANA planning, REFX, PPM, and analytical applications.

SAP Labs and SAP Australia delivery, March 2017 to November 2018.

Context

Stockland used Project and Portfolio Management and REFX modules on SAP S/4HANA, with BW Integrated Planning and SAP Analysis for Microsoft Office supporting financial planning, forecasting, and write-back processes.

Architecture Contribution

  • Developed ABAP and HANA-based BW-IP planning functions.
  • Built CDS views and AMDP objects to push processing into SAP HANA.
  • Supported ADSO read/write planning flows used by Analysis for Office.
  • Designed backend services for freestyle SAP UI5 applications using OData and CDS views.
  • Built Fiori applications with OData annotations and analytical applications using CDS/OData analytical annotations.
  • Implemented and optimised planning-function integration patterns, including Analysis for Office and VBA-triggered planning flows.

Impact

Connected S/4HANA real-estate and portfolio processes with planning, analytics, write-back, and Fiori user experiences, while improving performance through HANA code pushdown.
