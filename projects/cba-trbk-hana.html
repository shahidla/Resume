S/4HANA / Banking / Performance

Commonwealth Bank SAP TRBK HANA modernisation.

SAP Australia contractor, Mar 2025 to Nov 2025 and Jan 2026 to current.

Context

Supported CommBank's SAP Transactional Banking modernisation initiative as part of a core-banking transformation program. The work involved code remediation, performance optimisation, legacy issue resolution, and translation of DB2 query hints to HANA equivalents.

Architecture Contribution

  • Remediated custom code for efficient execution on HANA.
  • Adapted database-specific patterns from DB2 to SAP HANA.
  • Designed a data scrambling framework for S/4HANA non-production environments.
  • Implemented the framework using ABAP, CDS Views, and AMDP.
  • Built a Node.js MCP server exposing SAP RAP OData services as governed AI tools.

Impact

Reduced the data scrambling runtime from 240 hours to 6 hours, a 97% improvement, and delivered a scalable, stable framework for system copy and testing processes across the TRBK landscape.
